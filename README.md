### Costco 621 Location Analysis

This repository contains a quantitative analysis of retail fuel pricing trends and cost scenarios within SWFL.
Project Structure

    Data Collection/: Raw data and scripts used for gathering fuel pricing information.

    Data Exports/: Processed datasets and formatted analysis outputs.

    QAnalysis.ipynb: Jupyter Notebook featuring the data processing, visualizations, and quantitative models.

    Retail Fuel Pricing Report.pdf: A comprehensive summary report of the study's findings and regional insights.
